Realm has now existed for more than a decade, allowing local-first development across multiple platforms. In the process we have learned a lot about how the interplay between databases and programming languages changes in a local first world. I will talk a bit about lessons learned, and argue how it shows that much of what we today think of as databases really should be programming language features.
