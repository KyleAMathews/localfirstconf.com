Even apps that seem to defy local-first can hugely benefit from being built that way. We'll have a look at Succulent, a social media scheduling/analytics app that uses Jazz to make Instagram's API local-first. We'll see how easy it becomes to rapidly build features and decouple UI from backend tasks.
